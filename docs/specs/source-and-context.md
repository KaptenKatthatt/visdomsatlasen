# Sources and Context

## Purpose

Sources and historical context make Visdomsatlasen trustworthy, traceable and intellectually honest.

Every reflection room must be connected to identifiable source material.

The user should always be able to understand:

- where an idea comes from
- whether the room uses a quotation, translation, paraphrase or editorial interpretation
- who created the original material
- when and where it was created
- what historical context may affect its meaning

This information must remain available without dominating the first encounter with the reflection.

---

## Core Principle

> The idea comes first. The source remains close enough to verify.

Visdomsatlasen should not ask the user to accept a thought because it came from a famous person, sacred text or respected tradition.

At the same time, it must never detach ideas from their origins or present editorial writing as original source material.

---

## Relationship to the Reading Room

The Reading Room presents the reflection first.

Source information appears after the main text with lower visual emphasis.

The normal order is:

```text
Title
    ↓
Quiet opening
    ↓
Core reflection
    ↓
Thought to carry
    ↓
Questions in stillness
    ↓
Source
    ↓
Optional context
```

The source must be present on the page.

Detailed context may remain collapsed until the user chooses to open it.

---

## Source Visibility

Every published room must show a visible source reference.

The visible reference is deliberately minimal: the name of the author or,
when no author fits, the work. It is set in small letter-spaced capitals
with a downward caret, below the room's closing rule — a colophon line,
not a web link. Opening it reveals the details in place; it never
navigates away.

Example Swedish UI copy:

> EPIKTETOS ▾

Everything else — work, passage, editorial treatment, translation status,
uncertainty — is revealed when the reference is opened.

The name itself must not require an additional click.

---

## Source Detail

A secondary action may reveal fuller source information.

Suggested Swedish labels:

- Visa källan
- Om texten
- Historisk bakgrund

The expanded source detail may include:

- original title
- translated title
- author
- attributed author
- approximate date
- place or cultural context
- original language
- edition
- translator
- passage reference
- public URL or bibliographic reference
- copyright status
- editorial notes
- related uncertainty

---

## Types of Source Use

Every room must declare how it uses its source material.

Allowed types:

### Direct Quotation

The room includes an exact quotation from a specified edition or translation.

Required information:

- quotation marks or blockquote formatting
- exact source reference
- edition or translation
- translator, when applicable
- copyright status

Suggested Swedish label:

> Direkt citat

---

### Translation

The room includes a translation produced specifically for Visdomsatlasen.

Required information:

- original language
- translator or editorial responsibility
- whether the translation is literal or interpretive
- original source reference

Suggested Swedish label:

> Egen svensk översättning

Do not describe a loose interpretation as a translation.

---

### Paraphrase

The original idea is rewritten in new language while preserving its central meaning.

Required information:

- source reference
- clear statement that the wording is editorial
- no quotation formatting

Suggested Swedish label:

> Parafraserad återgivning

---

### Adaptation

The source has been substantially reshaped for a modern reflection room.

An adaptation may:

- combine several passages
- simplify historical language
- remove argumentation not needed for the room
- reorganise ideas
- connect the material to a modern human question

Suggested Swedish label:

> Bearbetad för reflektion

An adaptation must never be presented as the original author’s exact words.

---

### Editorial Reflection

The room is an original reflection inspired by one or more sources.

The source provides grounding, but the room’s central prose is written by the Visdomsatlasen editorial process.

Suggested Swedish label:

> Redaktionell reflektion inspirerad av källan

---

### Synthesis

The room combines ideas from several identified sources.

Required information:

- all substantial sources
- the relationship between them
- indication that the room is an editorial synthesis

Suggested Swedish label:

> Redaktionell sammanställning av flera källor

---

## Prohibited Source Presentation

Visdomsatlasen must never:

- place quotation marks around paraphrased text
- invent quotations
- attribute modern wording directly to an ancient author
- hide substantial source influences
- present AI-generated prose as historical source material
- imply certainty when authorship is disputed
- silently modernise the meaning of a source
- use a secondary quotation without checking the primary source when available
- cite only a quote website, social post or unattributed image

---

## Quotations

Direct quotations should be used sparingly.

The reflection room should not become a quotation collection.

A quotation is useful when:

- the original wording carries unique force
- the wording can be accurately sourced
- the passage is short enough to remain focused
- usage is legally permitted

Quotations must preserve meaningful context.

Do not extract a sentence in a way that reverses or distorts the original argument.

---

## Translation Policy

Translations must be treated as interpretations, not invisible replacements for the original.

When using an existing translation, record:

- translator
- publisher
- publication year
- edition
- relevant rights information

When Visdomsatlasen creates its own translation, record:

- original language
- source edition
- person or process responsible
- date of translation
- review status

Machine translation may support drafting.

It must not be published without human review.

---

## Historical Context

Historical context should help the user understand the text.

It should not become a general encyclopaedia article.

Useful context may explain:

- the historical period
- the intended audience
- political or religious circumstances
- important concepts that have changed meaning
- literary genre
- disputes about authorship
- differences between modern and historical assumptions

Context should be included only when it changes or deepens interpretation.

---

## Context Length

Context should remain proportional to the room.

Recommended lengths:

- visible source reference: one name
- short expanded context: 100–250 words
- extended source article: separate Library page

The Reading Room should not contain a long historical essay beneath every reflection.

Deeper material belongs in the Library.

---

## Uncertainty

Uncertainty must be stated clearly.

Examples include:

- disputed authorship
- uncertain dating
- fragmentary preservation
- several possible translations
- unclear cultural meaning
- later editorial additions
- legendary rather than historical attribution

Suggested Swedish wording:

> Textens exakta datering är osäker.

> Verket tillskrivs traditionellt Laozi, men författarskapet är omdiskuterat.

> Formuleringen kan översättas på flera sätt.

Uncertainty does not weaken the source.

Hiding uncertainty weakens trust.

---

## Religious Sources

Religious texts must be presented with the same care as philosophical and historical sources.

The application should distinguish between:

- what the source says
- how a tradition interprets it
- how the reflection room uses it

Avoid declaring theological claims as neutral historical facts.

Example:

> I Johannesevangeliet framställs Jesus som...

Prefer this over:

> Jesus var...

when the statement specifically describes the text’s theological presentation rather than independently established history.

---

## Mythological and Legendary Material

Myths and legends may contain valuable reflections.

They must be labelled according to their genre.

Suggested descriptions:

- myth
- sacred narrative
- legend
- literary dialogue
- traditional account
- historical chronicle

Do not present mythological events as verified history.

Do not dismiss them merely because they are not historical reporting.

The relevant question is what kind of source the user is encountering.

---

## Philosophical Sources

Philosophical sources should retain enough argumentative context to avoid reducing them to slogans.

A reflection may simplify an argument, but the source detail should clarify:

- the larger work
- the original philosophical problem
- important technical terms
- where the reflection departs from the original argument

Famous quotations must be verified against reliable editions.

---

## Oral Traditions

Oral traditions require particular care because they may not have one fixed original version.

Source records should identify:

- community or tradition
- collector or recorder
- approximate recording date
- publication
- language
- known variation
- whether the text is a later reconstruction

Avoid describing one recorded form as the single authentic version unless evidence supports that claim.

---

## Secondary Sources

Secondary scholarship may be used for:

- historical context
- interpretation
- dating
- authorship questions
- terminology
- translation comparison

Secondary sources should not replace the primary source when the room claims to represent a primary text.

Prefer:

- scholarly editions
- academic books
- peer-reviewed research
- museum or university publications
- established reference works

Popular sources may be useful for orientation but should not carry important factual claims alone.

---

## Source Hierarchy

When practical, source research should follow this order:

1. Primary source in the original language or a reliable critical edition.
2. Established translation.
3. Scholarly commentary or historical research.
4. Reputable institutional reference material.
5. General popular explanation.

Lower levels may help discovery.

They should not override stronger sources without a documented reason.

---

## Source Records

Each source should have one canonical record.

A source record may represent:

- a complete work
- a document
- a speech
- a letter
- a poem
- a scripture
- an inscription
- a recorded oral account
- an archaeological text

Specific passages should reference the canonical source record rather than duplicating source metadata.

---

## Suggested Source Model

```ts
type Source = {
  id: string;
  slug: string;
  title: string;
  originalTitle?: string;
  sourceType:
    | "book"
    | "scripture"
    | "letter"
    | "speech"
    | "poem"
    | "inscription"
    | "oral-tradition"
    | "historical-document"
    | "fragment"
    | "other";
  authorName?: string;
  attributedAuthorName?: string;
  traditionIds?: string[];
  originalLanguage?: string;
  approximateDate?: string;
  place?: string;
  description?: string;
  authorshipStatus?: "known" | "attributed" | "disputed" | "unknown";
  datingStatus?: "known" | "approximate" | "disputed" | "unknown";
  copyrightStatus: "public-domain" | "licensed" | "copyrighted" | "unknown";
  status: "draft" | "review" | "published" | "archived";
  createdAt: string;
  updatedAt: string;
};
```

---

## Suggested Passage Model

```ts
type SourcePassage = {
  id: string;
  sourceId: string;
  reference: string;
  originalText?: string;
  translatedText?: string;
  translator?: string;
  edition?: string;
  publicationYear?: number;
  sourceUrl?: string;
  notes?: string;
  status: "draft" | "review" | "published" | "archived";
};
```

---

## Room Source Relationship

A room may use one or several sources.

The relationship must state the type of use.

```ts
type RoomSourceRelation = {
  roomId: string;
  sourceId: string;
  passageId?: string;
  usageType:
    | "quotation"
    | "translation"
    | "paraphrase"
    | "adaptation"
    | "inspiration"
    | "synthesis"
    | "historical-context";
  isPrimary: boolean;
  editorialNote?: string;
};
```

At least one source relation must be marked as primary for every published room.

---

## Editorial Responsibility

Every room should have a clear editorial record.

Recommended fields:

```ts
type EditorialRecord = {
  roomId: string;
  writer?: string;
  sourceReviewer?: string;
  languageReviewer?: string;
  reviewedAt?: string;
  reviewNotes?: string;
  version: number;
};
```

These fields do not need to appear in the public interface.

They support accountability and revision.

---

## Source Review

Before publication, verify:

- source identity
- passage reference
- attribution
- quotation accuracy
- translation status
- relevant context
- copyright status
- uncertainty statements
- distinction between source and editorial prose

A room must not be published when a central source claim remains unverified.

---

## Copyright

Copyright status must be recorded before publication.

Public-domain source material may still require attribution.

A historical work may be public domain while a modern translation remains protected.

Do not assume that an online text is free to reuse because it is publicly accessible.

When rights are unclear:

- link to the source rather than reproducing it
- use a short legally permitted quotation where appropriate
- create an independently reviewed translation from a public-domain edition
- publish a paraphrase with clear attribution
- delay publication until status is resolved

---

## Source Links

External source links should be useful and stable.

Prefer links to:

- recognised digital editions
- libraries
- archives
- museums
- universities
- publishers
- scholarly databases

Avoid making the room dependent on one external page remaining available.

Store enough bibliographic information to locate the source again.

External links should open only after a deliberate user action.

---

## Library Presentation

The Library may provide a full source page.

A source page may contain:

- title
- origin
- date
- author or attribution
- historical context
- available passages
- connected reflection rooms
- translation information
- bibliographic references
- copyright status
- uncertainties

Source pages support exploration.

They must not imply that all available material has equal authority or historical certainty.

---

## AI-Assisted Content

AI may assist with:

- initial source discovery
- drafting summaries
- comparing wording
- structuring metadata
- identifying questions for human review

AI must not be treated as a source.

AI-generated claims must be verified against actual source material.

AI must never invent:

- quotations
- references
- page numbers
- editions
- translators
- historical context
- authorship conclusions

Published source information requires human editorial responsibility.

---

## Visual Design

Source information should be visually quieter than the core reflection but still readable.

Do not hide it through:

- extremely small text
- low contrast
- obscure icons
- hover-only interactions
- ambiguous labels

Source credibility should come from clarity, not decorative academic styling.

No emojis should be used.

---

## Mobile Experience

On mobile:

- the visible source summary should remain compact
- expanded context should open without losing the room position
- long bibliographic details may use a dedicated view
- external links should be easy to identify
- quoted text should not overflow horizontally
- the user should be able to return to the reflection easily

---

## Accessibility

Source and context features must support:

- semantic blockquotes
- correctly structured citations
- keyboard-accessible disclosure controls
- visible focus states
- screen readers
- sufficient contrast
- scalable text
- clear link descriptions

Do not use icons alone to distinguish quotations, translations and paraphrases.

---

## Acceptance Criteria

Sources and context are correctly implemented when:

- [ ] Every published room has at least one verified primary source.
- [ ] A visible source summary appears without requiring interaction.
- [ ] The room declares whether material is quoted, translated, paraphrased, adapted or editorial.
- [ ] Paraphrases are never formatted as quotations.
- [ ] Direct quotations include exact references and edition information.
- [ ] Modern translations are not assumed to be public domain.
- [ ] Historical uncertainty is stated clearly.
- [ ] Religious, mythological and philosophical sources are identified by source type and context.
- [ ] Detailed context remains optional in the Reading Room.
- [ ] Full source exploration is available through the Library.
- [ ] AI-generated claims are verified before publication.
- [ ] Source information remains accessible on mobile.
- [ ] No emojis are used.

---

## Out of Scope

The first version does not need:

- complete critical editions
- line-by-line scholarly commentary
- automatic translation comparison
- crowdsourced annotations
- public source discussions
- citation popularity metrics
- automated authority scores
- AI-generated historical reconstruction
- unsupported claims about original authorial intent

The goal is not to replace scholarship.

The goal is to make every reflection traceable, honest and understandable.
