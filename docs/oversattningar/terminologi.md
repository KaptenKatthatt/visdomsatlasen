# Terminologiordlista — testöversättningarna

Gemensam ordlista över de fyra passagerna (p1/p2/p3/p5) så att samma term återges
lika i hela atlasen. Bygger på `glm-5.2`:s terminologibeslut i flöde C
(`radata/<id>.json`), kontrollerade av Claude mot ankarna i
`docs/research/zen-experiment/utvardering/` och korsgranskningen. Avvikelser mellan
passager flaggas för ägarens beslut om enhetlig standard.

| Term | Läsning | Vald återgivning | Förekomst | Motivering / anmärkning |
|---|---|---|---|---|
| 衆生 | shujō | **varelser** (p1) / **levande varelser** (p3) | p1, p3 | *Avvikelse att harmonisera.* Rapporten och källkontexten använder "kännande varelser" (sanskr. *sattva*). Rekommendation: enhetligt **kännande varelser**, ev. kortas till "varelser" i vers av rytmiska skäl. |
| 仏 | butsu/hotoke | **Buddha** | p1, p3, p5 | Buddha-naturen i p1; behålls kategoriskt. |
| 仏法 | buppō | **buddhadharma / Buddhas dharma** | p3 | Teknisk term, inte generell "sanning". |
| 諸法 / 万法 | shohō / banpō | **(alla) dharmor** | p3 | "Dharmor" som fackterm framför "fenomen"; 万法 ("tiotusen dharmor") normaliseras till "alla dharmor" för parallellen. |
| 迷悟 | meigo | **illusion och upplysning** (p3) / **förvirring och upplysning** (utvärderingens ankare) | p3 | *Avvikelse:* "illusion" vs "förvirring" för 迷. Rekommendation: **förvirring och upplysning** (迷 = vilseföring, inte optisk illusion). |
| 生滅 | shōmetsu | **födelse och upphörande** | p3 | Korsgranskningen fångade riktigt att 滅 (upphörande) inte får bli "död" (死); *ordagranna* versionen korrekt, den läsbara rättas. |
| 生仏 | shōbutsu | **(levande) varelser och buddhor** | p3 | Sammandragning av 衆生 + 諸仏 — inte 成仏 ("att bli buddha"). |
| 豊倹 | hōken | **överflöd och brist** | p3 | Inte "fullhet och tomhet"; poetiskt motsatspar/metonymi för dualiteter. |
| 有時 | uji / yūji | **varande-tid** (Dōgens begrepp) / **ibland** (kanbun-citatet) | p5 | Dubbeltydigheten hålls synlig; citatet läses adverbialt ("ibland"), kommentaren filosofiskt ("varande-tid"). Kärnan i passagen. |
| 丈六(金身) | jōroku (konjin) | **sexton fot (den gyllene gestalten)** | p5 | ~4,8 m; inte "sex fot". 八尺 = åtta fot (~2,4 m) — två gestalter/mått. |
| 払子 | hossu | **viftare** (flugviska) | p5 | Ceremoniell flugviska — inte "barn", "kvast" eller "flugsmälla". |
| 露柱 / 灯籠 | rochū / tōrō | **bar pelare** / **lykta** | p5 | Två ting, får inte slås ihop. |
| 虚空 | kokū | **tomma rymden / rymden** | p5 | Inte "intet". |
| 鉢盂 | hatsu-u / boyu | **munkskål** (allmoseskål) | p2 | Konkret föremål; inte pseudoetymologisk "almskål". |
| 有省 | yūshō | **fick insikt** | p2 | Etablerad zenterm för plötslig förståelse/uppvaknande. |
| 叢林 | sōrin / conglin | **klostersamfund** | p2 | Institutionell/andlig betydelse framför bokstavlig "tät skog". |
| 頌 | ju / song | **oden** (vers) | p2 | Poetisk kommentar i zenlitteratur. |
| 開口見膽 / 露出心肝 | — | (idiom) "visar sitt innersta" | p2 | *Redaktionell not:* de konkreta organen 膽 (galla/mod), 心肝 (hjärta/lever) bör bevaras tydligare än glm:s "hjärta"/"innersta" (korsgranskningens riktiga fynd). |
| 本来 | honrai | **ursprungligen** | p1 | Anspelar på *hongaku* (ursprunglig upplysning) utan tolkande omskrivning. |
